Skriv med egne ord hvad display pro-perty påvirker? Den påvirker hvordan elementet vises, f.eks. hvor stor del plads den tager i en container og om det er usynligt.
Hvad gør display: block? A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
Hvad gør display: inline? An inline element does not start on a new line and only takes up as much width as necessary.
Hvilke fordele er der ved display: in-line-block Sammenlignet med ”display: inline” er den største forskel, at ”display: inline-block” gør det muligt at indstille en ”width” og ”height” på elementet. Også med display: inline-block, top og buttom respekteres margin / padding, men display: inline gør ikke. Sammenlignet med display: block, er den største forskel er, at display: inline-block ikke tilføjer et line-break efter ele-mentet, så elementet kan sidde ved siden af andre elemen-ter.
