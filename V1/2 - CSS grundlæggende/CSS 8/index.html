Med egne ord: hvad påvirker position-property? 
                Kort sagt bruges det til at flytte elementer rundt så man kan placere det som man vil have på sin hjemmeside. Lidt mere i detalje:
                Position-property angiver den type positio-neringsmetode, der bruges til et element Der er fem forskellige position-values: 
                •	static
                •	relative
                •	fixed
                •	absolute
                •	sticky
                Elements bliver positioneret ved at bruge properties: 
                •	top
                •	bottom 
                •	left
                •	right
                Disse egenskaber fungerer dog ikke, med-mindre position-property er angivet først. De fungerer også forskelligt afhængigt af position-value.
Hvad gør position: relative? 
                    Den bruges til at flytte sit ”element” i forhold til den position den vil være på i ”static”.
Hvad positionerer absolut-positionerede elementer sig i forhold til? 
                    Hvis Relative ikke er angivet:   if an abso-lute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.

                    Hvis relative er angivet: An element with position: absolute; is posi-tioned relative to the nearest posi-tioned ancestor (instead of posi-tioned relative to the viewport, like fixed).
